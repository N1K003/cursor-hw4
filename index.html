Task 2

Please, send us explanation why console.log(1); setTimeout(() => { console.log(2) }, 0); console.log(3);
will print 1,3,2? Please, read about EventLoop at JavaScript.
Timers (setTimeout) is just a planning mechanism which is creating task to be executed after period of time
(not exactly in).
So, we are:
  1. executing task console.log(1)
  2. panning task setTimeout(() => { console.log(2) }, 0)
  3. executing next task console.log(3)
  4. executing planned task console.log(2)
